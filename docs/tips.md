# tips

* 2.分清楚字符串和整数，浮点数的区别  
* 3.搞清楚循环的结构和结果  
* 4.根据题目作答，审清楚题意  
* 5.灵活运用注释，让自己的代码更加简洁易懂  
* 6.在计算总和时记得初始化变量 
* 7.记得检查一遍，括号(和扩号)是对称的，就是有几个向左的括号就有几个向右的括号
