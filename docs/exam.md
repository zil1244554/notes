# 测试题  
## 一卷
编写程序代码，实现以下功能：  
* (1)将数值9.7赋值给a；  
* (2)将数值4.6赋值给变量b；  
* (3)将a减b的值赋值给c；  
* (4)求a,b,c的最大值赋值给s；  
* (5)使用输出语句在屏幕上输出s的值  


## 二卷

1

3

6
